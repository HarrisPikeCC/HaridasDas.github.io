Overview

The Mozilla Developer Network defines an element as "is a part of a webpage." They continue that "A typical element includes an opening tag with some attributes, enclosed text content, and a closing tag."

This document defines some of the most common HTML elements.

Element Tags

Main Root

Top level element of a document.

Tag Name Description
html HTML "Represents the top-level element of an HTML document... All other elements must be descendants of this element."1

Document metadata

"Metadata contains information about the page... Metadata for styles and scripts may be defined in the page or link to another file that has the information."1

Tag Name Description
head head "Contains general information about an HTML page that isn’t displayed on the page itself."2
link link "Specifies relationships between the current document and an external resource. This element is most commonly used to link to CSS" 1
style style Used for putting CSS directly into the document.

Sectioning

"Sectioning elements allow you to organize the document content into logical pieces. Use the sectioning elements to create a broad outline for your page content, including header and footer navigation, and heading elements to identify sections of content."1

Tag Name Description
body body "Represents the content of an HTML document. There can be only one body element in a document.1
header header "Describes the content at the top of the page."2
footer footer "Includes the page’s footer content at the bottom of the" page.2
